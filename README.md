## Installation

> For building from source, see [docs/BUILD.md](docs/BUILD.md).

## Usage

```bash
ncea-cli
```

Full usage, configuration, and troubleshooting: [docs/DOCS.md](docs/DOCS.md).

## Features

- Fuzzy search over titles, subjects, and keywords
- Multi-source retrieval with source preference and override support
- Interactive year and paper-type selection
- Manifest and adapter caching for faster repeat lookups
- Cache and manifest maintenance commands
